# 🚀 LangChain - End-to-End Learning & AI Application Development

A comprehensive hands-on repository demonstrating the core concepts of **LangChain** through practical notebooks and real-world AI applications.

This project covers the complete LangChain workflow, from building simple prompt chains to creating intelligent AI agents with tools, memory, middleware, and structured outputs.

---

# 📌 Project Overview

This repository is designed to help developers understand and implement modern **LLM-powered applications** using LangChain.

The notebooks are organized progressively, allowing learners to move from beginner concepts to advanced agentic AI workflows.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sripada-Sudheer28/LangChain.git
```

Navigate to the project

```bash
cd LangChain
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
OPENAI_API_KEY=your_key
GROQ_API_KEY=your_key
GOOGLE_API_KEY=your_key
ANTHROPIC_API_KEY=your_key
```

Run notebooks using Jupyter Notebook or VS Code.
